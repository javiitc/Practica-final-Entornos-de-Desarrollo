# DAM - 0487. Entorns de Desenvolupament: Práctica Final RA1

## Descripción
Nuestro código en Java ofrece cinco programas diferentes. Ha sido un proyecto final, en equipo, para nuestra clase de Entornos de Desarrollo, donde hemos utilizado todos los conceptos que sabemos hasta el momento (Scanner, condicionales, conversiones, bucles, etc.)
## Características
El programa empieza con un menú que permite a los usuarios elegir entre los cinco programas que se pueden ejecutar: 

1. **Calculadora Básica** - Realiza operaciones de matemáticas básicas (suma, resta, multiplicación, división)
2. **Clasificador de Números** - Determina si un número es par/impar y positivo/negativo
3. **Validador de Contraseñas** - Verifica la fortaleza de contraseñas basándose en los requisitos
4. **Generador de Números Aleatorios** - Genera números aleatorios dentro de un rango especificado por el usuario
5. **Contador de Palabras y Caracteres** - Cuenta el total de caracteres y palabras en un texto ingresado

## Cómo Usar
1. Ejecutar el programa
2. Seleccionar una opción del menú (1-5)
3. Seguir las instrucciones para la función elegida
4. El programa se ejecuta y mostrará los resultados
