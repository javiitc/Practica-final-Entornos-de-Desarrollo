# DAM - 0487. Entorns de Desenvolupament: Práctica Final RA1

## Descripción
Hicimos un código en Java que ofrece cinco programas de utilidad diferentes en uno. Desarrollado como proyecto en equipo (de dos), demuestra conceptos fundamentales de programación incluyendo manejo de entrada de usuario (Scanner), condicionales, bucles y manipulación de cadenas.

## Características
El programa incluye un sistema de menú que permite a los usuarios elegir entre los cinco códigos diferentes que realizamos:

1. **Calculadora Básica** - Realiza operaciones de matemáticas (suma, resta, multiplicación, división)
2. **Clasificador de Números** - Determina si un número es par/impar y positivo/negativo
3. **Validador de Contraseñas** - Verifica la fortaleza de contraseñas basándose en los requisitos
4. **Generador de Números Aleatorios** - Genera números aleatorios dentro de un rango especificado por el usuario
5. **Contador de Palabras y Caracteres** - Cuenta el total de caracteres y palabras en un texto ingresado

## Cómo Usar
1. Ejecutar el programa
2. Seleccionar una opción del menú (1-5)
3. Seguir las instrucciones para la función elegida
4. El programa se ejecuta y mostrará los resultados

## Desarrollo en Equipo
En equipos de dos personas, desarrollamos un programa que incluye cinco de las diez opciones especificadas por el profesor. Al iniciar, el programa presentará un menú que permitirá seleccionar cualquiera de estas opciones.
